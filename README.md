# Pitch Perfect 🔊

This app lets you record your voice using AVFoundation and play it back in peculiar ways:

* Slow 🐌
* Fast 🐇
* High Pitch 🐿
* Low pitch 🐘
* Echo 🐦
* Reverb 🔊
